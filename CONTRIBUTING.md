# Contributing to N.O.M.O.R.E.

Mission first: Protect kids, ship reports, stay legal.

## Rules of Engagement
- Public data only. No paywalls, no fake accounts, no CFAA violations.
- All code must hash/delete – never store CSAM.
- PRs: Feature branches, descriptive commits (e.g., "feat: add BrandArmy scraper").
- Tests required for scrapers/classifiers.
- Sign off on legal: "I certify this complies with 18 USC § 2258A."

Bounties via OpenCollective. Hunt smart.
